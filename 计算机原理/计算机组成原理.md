## 命令或工具

### gcc编译并查看

```shell
$ gcc -g -c test.c
$ objdump -d -M intel -S test.o
$ readelf -s link_example.o //查看符号表
$ objdump -r link_example.o //查看重定位表
```

### gcc编译动态链接库

```shell
$ gcc lib.c -fPIC -shared -o lib.so //-fPIC：Position Independent Code 地址无关
$ gcc -o show_me_poor show_me_poor.c ./lib.so
```
